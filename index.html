Python - Mac下PyCharm&PyQt5环境搭建

发表于 2020-08-25

macOS下PyQt-tools的搭建折腾了很久，发现python的包没有macOS版本的包，找了好久，终于装上了…

Qt

需要提前装好Qt，首先，进入QT官网选择需要的版本， 我选的是qt-opensource-mac-x64-5.14.1.dmg，下载安装完成之后就好了。这个安装包不知道为什么，我就是下载不下来，用了各种方法，最后还是找了个国外的朋友帮我下下来的…

PyCharm

配置 QtDesigner

这是个Qt的界面可视化工具，可以很方便的设计UI，所见即所得。Designer的位置要找一下，我的是在/usr/local/Cellar/qt/5.14.1/libexec/Designer.app。
我们可以在PyCharm中配置Designer的路径，这样就可以在PyCharm中快速启动Designer了。

  1. 打开 PyCharm 的设置
  2. 找到 Tools -> External Tools
  3. 点击 + -> Create Tool
  4. 配置中把「工具设置」中的『程序』填Designer的位置，工作目录填$ProjectFileDir$即可

配置 PyUIC

这个工具可以把Qt Designer的UI文件，转换成Py文件。

  1. 打开 PyCharm 的设置
  2. 找到 Tools -> External Tools
  3. 点击 + -> Create Tool
  4. 配置中把「工具设置」中的『程序』填你的python解释器路径，Parameters：-m PyQt5.uic.pyuic $FileName$ -o $FileNameWithoutExtension$.，工作目录填$ProjectFileDir$即可

使用

右键菜单中选择 External Tools 即可
